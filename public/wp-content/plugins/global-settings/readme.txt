== Description ==

Global Setting allows you to add, update and delete the global variables for your blog. Get Paid version at : http://just4u.x10.bz/global

Features include: 

1. You can add new custom global variables for your blog.

2. Short code available to use that created option values in post and page.

3. You can add Logtext as global variable value with HTML tags.

4. You can add Global settings for your site i.e Facebook Api Key, Home page text etc.



== Settings ==

Go to Admin -> click on Global Setting tab.

1. Click on add option button to add new option.

2. Duble click on option value to edit it and then press update button to save it.
